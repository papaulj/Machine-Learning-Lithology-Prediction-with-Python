This project aims to develop a machine learning model that predicts lithology (either reservoir rock or non-reservoir rock) from well logs using Gamma ray logs, DPHI porosity, Neutron porosity, caliper logs, and Deep Induction log¶

Real well log data was obtained from Kansas field and available at https://kgsimages.blob.core.windows.net/web/web_1/WebDocs/WellLogs/kcc_logs_2025/1056890332.las¶

6. Interpretation of model prediction result¶

The classification report evaluates the performance of the machine learning model by comparing predicted lithology against the true labels. It reports metrics such as precision, recall, and F1-score for each class. In my case the model achieved perfect accuracy, which likely occurs because the lithology labels were derived from the gamma ray log, which is also included as an input feature.

The above graph reveals that gamma ray is the most significant predictor of the lithology type. This is not unexpected due to the inherent use of gamma ray values to differentiate between lithologies. The second most importnt determinant of the lithology type is neutron porosity and then followed by deep induction logging values of the interval.

9. Interporetation and Conclusion¶

Machine learning classification was used to predict lithology from well log measurements. The Random Forest model was the algorithm selected due to its published woderful performance on geologic dataset. The developed model achieved strong predictive performance, with Gamma Ray emerging as the most influential feature. The predicted lithology intervals align well with conventional log interpretation, and consequently proves reliable.
